Для установки службы необходимо:
1) Изменить (если нужно) TCP-порт в _install.cmd (параметр: "-port X"). По умолчанию установлен 81
2) Запустить под администратором _install.cmd
3) Разрешить в Firewall'е установленный порт или %SystemRoot%\knlmsvc.exe

Для удаления службы:
1) Запустить под администратором _uninstall.cmd
2) Удалить %SystemRoot%\knlmsvc.exe
