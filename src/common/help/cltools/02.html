Браузер


Безопасный браузер Internet представляет собой замену стандартного IE для безопасной и удобной работы пользователя.
Безопасный IE включается/отключается на вкладке "Браузер" настроек шелла.
Такие функции как сохранение страницы, загрузка файлов и пр. интегрированы со средой шелла.

Также есть возможность запуска в простом режиме (т.е. без адресной строки и кнопок).
Такой режим может использоваться для просмотра локальных страниц новостей клуба и пр.
Для этого необходимо просто запустить bodywb.exe с параметром -simple:
bodywb.exe -simple http://www.runpad-shell.com
Иногда бывает полезным запускать браузер в режиме отказа от запретов на сайты.
В этом случае просто запустите его с параметром -r:
bodywb.exe -r
