Проводник пользователя


Безопасный проводник пользователя представляет собой замену стандартного проводника для безопасной и удобной работы пользователя.
Безопасный проводник доступен из меню пользователя "Мой компьютер", а большая часть его настроек доступна на вкладках "Файлы", "Проводник" настроек шелла.
Благодаря ему пользователь может удобно работать с дискетой, flash-диском, CD-ROM, своей папкой пользователя и дополнительными папками, установленными администратором. Таким образом пользователь не может "зайти" внутрь файловой системы компьютера, однако администратор может установить дополнительные папки для копирования файлов в/из некоторых папок. Также доступна опция удобного архивирования файлов с помощью WinRAR, работа с Bluetooth
Запуск многих файлов также ограничен в целях безопасности. На вкладке "Безопасность: Файлы" администратор может сам устанавливать разрешенные расширения запускаемых файлов.

Также можно запускать проводник с параметрами:
Открыть ресурс "Дискета":
bodyexpl.exe $diskette
Открыть ресурс "Flash":
bodyexpl.exe $flash
Открыть ресурс "CD-ROM":
bodyexpl.exe $cdrom
Открыть ресурс "VIP-папка":
bodyexpl.exe $vipfolder
Открыть ресурс "Папка пользователя":
bodyexpl.exe $userfolder
